CARPETA DE AVATARES
===================

Coloca aquí las imágenes de los participantes.

Formato recomendado: PNG o JPG cuadrado, mínimo 128×128px.
Nombre de archivo: debe coincidir con el campo "avatar" de cada participante en script.js.

Ejemplo:
  avatars/el-profe.png
  avatars/la-bruja.png

Mientras no haya imágenes, la web muestra círculos de color con la inicial del alias.
